# asset_repacker

A library providing a runtime asset repacker that allows other mods to contribute asset patches and custom content to the game while reducing the risk of conflicting patches.

**NOTE:** This library is in early stages of development and may change drastically over the following versions. If you would like to see an example of how to use this library, for now please see [the source of Dinomod Enhanced: Recompiled](https://github.com/EoinODoodles/dinomod-enhanced-recompiled).
